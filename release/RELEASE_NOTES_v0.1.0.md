# SkillFlow Mac v0.1.0 Release Notes

## 中文

### 概览
SkillFlow Mac 首个公开发布版本，提供技能扫描、预览与同步能力，并保持全中文界面体验。

### 发布策略（重要）
- 本版本仅发布 macOS 资产。
- 不提供 Windows / Linux 安装包。

### 下载资产（macOS）
- `SkillFlow-Mac_0.1.0_aarch64.app.zip`
- `SkillFlow-Mac_0.1.0_aarch64.dmg`

### 校验
下载后可执行：

```bash
shasum -a 256 SkillFlow-Mac_0.1.0_aarch64.app.zip
shasum -a 256 SkillFlow-Mac_0.1.0_aarch64.dmg
```

参考校验值见：`release/macos/v0.1.0/SHA256SUMS.txt`

### 安装建议
- 普通用户优先使用 `.dmg` 安装。
- 开发/分发场景可使用 `.app.zip`。
